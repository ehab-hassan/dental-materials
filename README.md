# DentalPro — Dental Materials Website

A modern, professional marketing website for a dental materials company (established 1986). Built with **Next.js**, **Tailwind CSS**, and a mobile-first responsive layout.

## Tech Stack

- **Next.js** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **React** 19

## Features

- **Hero**: Full-width slider (3 slides), fade transition, prev/next + dots, dark overlay, CTA
- **Navigation**: Sticky, transparent over hero, solid background on scroll, mobile menu
- **Trust**: Stat cards (Years, Products, Clients, Certifications)
- **Product categories**: 4 hover cards with icons
- **Why Choose Us**: Icon-based benefits, medical layout
- **Featured products**: Product cards with hover and “Add to cart” (UI only)
- **CTA**: Strong call-to-action block
- **Footer**: Minimal, professional

## Design

- Medical blue + white
- Accent orange for primary buttons
- Clean spacing and premium typography (DM Sans)
- Light animations and hover states

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm start
```

## Project Structure

- `src/app/` — App Router (layout, page, globals.css)
- `src/components/` — Navbar, HeroSlider, TrustSection, ProductCategories, WhyChooseUs, FeaturedProducts, CTASection, Footer
